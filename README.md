# Moyenne Mediane

Game Chasing Québec, le répertoire des adresses de boutiques de jeux vidéos dans la province.

## Installation

Installer Game Chasing Québec

```bash
yarn install
```

## Lancer l'application

Démarrer le projet

```bash
yarn run dev
```
